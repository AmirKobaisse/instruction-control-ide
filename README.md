# Instruction Control IDE

**Desktop IDE for creating, running, and debugging custom instruction files using Python.**

---

## Overview

Instruction Control IDE is a desktop application built in Python for creating, editing, and executing custom instruction files. 
It provides a GUI editor, run functionality, and a framework for adding an auto-debugger. The IDE is designed for local execution 
and future integration with custom hardware.

---

## Features

- GUI-based text editor for instruction files
- Run button to execute instructions
- Output console for feedback and results
- Placeholder for auto-debugging of syntax errors
- Modular Python package structure for easy expansion

---

## Tech Stack

- Python 3.10+
- Tkinter (GUI)
- Future support for MySQL or other storage options
- Designed for local desktop execution
